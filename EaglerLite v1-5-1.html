Optimization Details

Context Tuning

Adjusts getContext options once before canvas creation. Zero runtime interception risk.

Background Throttle

Pauses the game loop to 1 FPS when unfocused. Reduces CPU usage by 90%+ when alt-tabbed.

Flat Textures

Forces nearest-neighbor filtering. Greatly improves FPS on integrated graphics without changing blocky aesthetics.

Mipmap Suppression

Skips generation of low-res texture pyramids. Saves CPU cycles during atlas stitching and saves VRAM.

Disable Dithering

Turns off GL_DITHER. Invisible on modern screens but saves GPU fill-rate computations per pixel.

Stutter Fix

Orphans VBO memory before sub-updates so the CPU never waits for the GPU. Eliminates block-break lag.

Uncap FPS

Bypasses browser V-Sync using MessageChannel to allow 100-300+ FPS without micro-stuttering.

Audio Offloading

Tells the browser to prioritize smooth audio playback over low latency, saving main thread CPU cycles.

No Colorspace Conversion

Skips OS-level sRGB math during texImage2D. Speeds up atlas stitching and texture loading.

Suppress GL Errors

Bypasses getError() queries. Prevents the CPU from stalling while synchronizing with the GPU.

Disable Line Smoothing

Turns off LINE_SMOOTH. Saves GPU edge-blending calculations on wireframes and block outlines.

Single-Sampled FBOs

Forces renderbufferStorage over Multisample. Saves VRAM and skips offscreen MSAA resolving.

CSS Canvas Compositing

Applies pixelated rendering to the canvas. Skips OS-level CSS smoothing and compositing math.

Quality-of-Life Details

Fullbright

Verifies the VBO is exactly 28 bytes per vertex (1.12.2 terrain format) before mutating the lightmap shorts to 208. Clouds and entities are ignored.

AutoSprint

Reads your custom keybind and dispatches it seamlessly. Utilizes vanilla sprint mechanics to ensure 100% reliable sprinting.

Zoom

Applies a pure CSS scale(4) transform to the canvas. Avoids WebGL matrix math entirely, preventing bending or X-ray.
